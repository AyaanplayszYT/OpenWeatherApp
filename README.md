## Features

- Current weather and 7-day forecast
- Hourly breakdown
- Weather alerts
- Air quality and pollen info
- UV index and historical weather
- Location search and favorites
- Dark mode


## Getting Started

1. **Clone the repository**
	```bash
	git clone https://github.com/AyaanplayszYT/OpenWeatherApp.git
	cd OpenWeatherApp
	```

2. **Install dependencies**
	```bash
	npm install
	```

3. **Add your OpenRouter API key**
	- Create a `.env` file in the project root
	- Add this line:
	  ```
	  VITE_OPENROUTERKEY=your_key_here
	  ```

4. **Add a background image**
	- Place your image at `public/background.jpg`

5. **Start the development server**
	```bash
	npm run dev
	```
	- Open your browser at `http://localhost:5173/`
